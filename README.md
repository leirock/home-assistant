Home Assistant Configuration
======================
[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)

This repository contains the necessary files for integrating the KNX lighting system of Gemdale Greater City Bay into the Home Assistant platform, and enables voice control of the lights via the Xiaomi Xiaoai speaker.
